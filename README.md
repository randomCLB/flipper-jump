# Flipper Jump 🐬

A Flappy Bird-style game with a dolphin swimming in the ocean!

## How to Play
- Click or press Space to make the dolphin jump
- Avoid the coral pipes
- Score points as you survive!

## How to Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g., "flipper-jump")

2. Upload these files:
   - `index.html`
   - (optional) any image files you add

3. Go to repository Settings → Pages

4. Under "Build and deployment":
   - Source: Deploy from a branch
   - Branch: main (or master)
   - Folder: / (root)

5. Save and wait ~1-2 minutes

6. Your game will be live at: `https://yourusername.github.io/flipper-jump/`

## Controls
- **Click** or **Tap** - Jump
- **Space** - Jump (desktop)
